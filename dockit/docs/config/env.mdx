# 环境变量

| 变量 | 作用 | 示例 |
| ---- | ---- | ---- |
| `PVM_HOME` | 覆盖 PVM 根目录（默认 `~/.pvm`） | `PVM_HOME=/opt/pvm` |
| `PVM_NODE_VERSION` | 临时覆盖 node 版本（最高优先级） | `PVM_NODE_VERSION=18.19.0` |
| `PVM_PYTHON_VERSION` | 临时覆盖 python 版本 | `PVM_PYTHON_VERSION=3.11` |
| `PVM_GO_VERSION` | 临时覆盖 go 版本 | `PVM_GO_VERSION=1.21` |

## 解析优先级回顾

环境变量 > 项目 `.pvmrc` / `.nvmrc` > 用户全局 `~/.pvm/versions` > 系统 PATH。

## 示例

```bash
# CI 中临时固定版本
export PVM_NODE_VERSION=20.11.0
pvm use node
node -v   # v20.11.0
```

:::info
环境变量名规则为 `PVM_<运行时大写>_VERSION`，运行时名取 `SupportedRuntimes` 中的小写名。
:::
