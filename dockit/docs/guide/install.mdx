# 安装与初始化

## 下载二进制

从发布页下载对应平台的二进制，放到 PATH 中即可，无需安装器：

| 平台 | 文件 |
| ------ | ---- |
| Windows | `pvm-windows-amd64.exe` / `pvm-windows-arm64.exe` |
| macOS | `pvm-darwin-amd64` / `pvm-darwin-arm64` |
| Linux | `pvm-linux-amd64` / `pvm-linux-arm64` |

Windows 用户也可使用 MSI 安装包 `pvm-windows-amd64.msi` 完成注册。

## 首次初始化（setup）

首次直接运行 `pvm`（不带任何参数）会自动触发初始化：

```bash
pvm
# Welcome to pvm! It looks like this is your first run.
# Running setup to configure pvm...
```

手动初始化：

```bash
pvm setup
```

`setup` 会完成：

- 创建根目录 `~/.pvm`（可用环境变量 `PVM_HOME` 覆盖）。
- 创建 `versions/`、`installs/`、`shims/` 子目录。
- 把 `~/.pvm/shims` 与 `~/.pvm/bin` 加入用户 PATH。

## 检查 PATH

若终端找不到 `pvm` 或 `node` 等命令，运行自检：

```bash
pvm setup-path
```

该命令会检查并修复 PATH 配置、提示与其它版本管理器的冲突。

## 验证安装

```bash
pvm version
# pvm 0.0.0
```

```bash
pvm doctor
# 检查 pvm 安装健康度（目录、PATH、shims 完整性）
```

:::warning 注意
`setup` / `setup-path` 修改的是**用户级** PATH，无需管理员权限。
修改后请**重启终端**（或 IDE）让 PATH 生效。
:::
