# 安装与初始化

## 一键安装

| 平台 | 命令 |
| ---- | ---- |
| Windows (PowerShell) | `irm https://raw.githubusercontent.com/Lycorxy/pvm/main/scripts/install.ps1 \| iex` |
| macOS / Linux | `curl -fsSL https://raw.githubusercontent.com/Lycorxy/pvm/main/scripts/install.sh \| bash` |

## 手动下载

PVM 是**单一二进制**（不含额外的 shim 文件）。从 [Release 页](https://github.com/Lycorxy/pvm/releases) 下载对应平台的文件，放到 PATH 中即可：

| 平台 | 文件 | 直接下载 (latest) |
| ---- | ---- | ---------------- |
| Windows x64 | `pvm-windows-amd64.exe` | [pvm-windows-amd64.exe](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-windows-amd64.exe) |
| Windows ARM64 | `pvm-windows-arm64.exe` | [pvm-windows-arm64.exe](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-windows-arm64.exe) |
| macOS Intel | `pvm-darwin-amd64` | [pvm-darwin-amd64](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-darwin-amd64) |
| macOS Apple Silicon | `pvm-darwin-arm64` | [pvm-darwin-arm64](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-darwin-arm64) |
| Linux x64 | `pvm-linux-amd64` | [pvm-linux-amd64](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-linux-amd64) |
| Linux ARM64 | `pvm-linux-arm64` | [pvm-linux-arm64](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-linux-arm64) |

也可下载 `pvm-vX.X.X-{os}-{arch}.zip` 解压后使用。

Windows 还提供 MSI 安装包（双击安装，自动配置 PATH 并运行 `pvm setup`）：

| 平台 | MSI 安装包 |
| ---- | ---------- |
| Windows x64 | [pvm-windows-amd64.msi](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-windows-amd64.msi) |
| Windows ARM64 | [pvm-windows-arm64.msi](https://github.com/Lycorxy/pvm/releases/latest/download/pvm-windows-arm64.msi) |

:::tip 仓库地址
源码：[github.com/Lycorxy/pvm](https://github.com/Lycorxy/pvm) ·
Release：[github.com/Lycorxy/pvm/releases](https://github.com/Lycorxy/pvm/releases)

## 首次初始化（setup）

首次直接运行 `pvm`（不带任何参数）会自动触发初始化：

```bash
pvm
# Welcome to pvm! It looks like this is your first run.
# Running setup to configure pvm...
```

手动初始化：

```bash
pvm setup
```

`setup` 会完成：

- 创建根目录 `~/.pvm`（可用环境变量 `PVM_HOME` 覆盖）。
- 创建 `versions/`、`installs/`、`shims/` 子目录。
- 把 `~/.pvm/shims` 与 `~/.pvm/bin` 加入用户 PATH。
- 如果系统 PATH 中有 nvm/node/python 等冲突路径，自动把 pvm shims **前置到系统 PATH 最前**（需管理员权限），确保 pvm 的版本优先生效。

## 检查 PATH

若终端找不到 `pvm` 或 `node` 等命令，运行自检：

```bash
pvm setup-path
```

该命令会检查并修复 PATH 配置、提示与其它版本管理器的冲突。

## 验证安装

```bash
pvm version
# pvm 0.0.1
```

```bash
pvm doctor
# 检查 pvm 安装健康度（目录、PATH、shims 完整性）
```

:::warning 注意
修改 PATH 后请**重启终端**（或 IDE）让 PATH 生效。
如果系统 PATH 冲突修复需要管理员权限，setup 会提示 UAC 弹窗。
:::
