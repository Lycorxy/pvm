# self-update / uninstall

## self-update

升级 PVM 自身到最新版本。

```bash
pvm self-update
```

升级后**自动运行 `reshim`**——因为 pvm 二进制已替换为新版本，shims（硬链接）仍指向旧 inode，
必须 reshim 才指向新二进制。如果 reshim 失败（如某些 shim 正在运行），会在下次 `pvm use` 或 `pvm install` 时自动重建。

## uninstall

从当前机器卸载 PVM。

```bash
pvm uninstall
```

会安全移除各版本目录（adopted 的 junction 版本只删链接不删目标），清理 `~/.pvm` 目录，
并提示从 PATH 中移除相关条目（该步需重启终端生效）。

:::warning 注意
`uninstall` 不会删除你已安装的运行时二进制所在目录之外的用户数据，请确认后执行。
:::
