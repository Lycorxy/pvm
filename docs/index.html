PVMPolyglot Version Manager
基于 shim 驱动的多语言版本管理器

一个工具
管好所有运行时

node · python · go · git · pnpm · yarn
基于 shim 驱动，无需管理员权限，项目级版本隔离

快速开始 命令参考配置说明
bash
# 安装并锁定项目版本
$ pvm install node@20.11.0
$ pvm use node@20.11.0
# ✓ 已切换到 node 20.11.0

核心能力

从安装到隔离，一条命令贯穿研发全流程

一键安装

一条 install 装好 node、python、go 等，默认走国内镜像，无需逐个配置源

秒级切换

pvm use 按项目 / 用户 / 系统三级作用域切换，多版本互不污染

项目隔离

.pvmrc 声明版本并提交，团队 clone 后 pvm use 即得一致环境

无需 sudo

运行时全部装在用户目录 ~/.pvm，免管理员权限，CI / 容器内同样可用

Shim 驱动

通过 shim 拦截命令，按需解析版本，shell 启动零开销、切换即时生效

镜像加速

默认国内镜像下载，--official 可切回官方源，--mirror 可强制镜像

当你敲下 node，PVM 如何决定版本

shim 拦截命令后，按 4 级优先级自高向低解析，命中即生效

1

环境变量

最高

PVM_NODE_VERSION=20.11.0，临时覆盖，优先级最高

2

.pvmrc / .nvmrc

从当前目录向上查找，落地项目级版本约定

3

用户全局

~/.pvm/versions 中 pvm use 写入的默认版本

4

系统 PATH

兜底

系统已装同名命令，作为兜底不下载

3 步上手

从零到项目级版本锁定，几分钟即可完成

01

安装 PVM

一条脚本装好命令行工具，并写入 PATH

curl -fsSL https://pvm.sh/install | sh
02

安装运行时

按需安装任意版本，默认走国内镜像

pvm install node@20.11.0 python@3.12.0
03

进入项目即生效

声明版本并提交，团队 clone 后即可一致运行

pvm use node@20.11.0
# 写入 .pvmrc，提交到仓库
